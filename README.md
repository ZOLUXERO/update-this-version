# update-this-version
Update this version using Semantic versioning

```
npm i -g update-this-version
npm i -g update-this-version --release
git add .
git commit -m "chore(release): version [skip ci]"
git push -u origin main --tags
```

## usage:
```
update-this-version
node ./cli.js
node ./cli.js --release
```
